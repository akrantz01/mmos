Client¶

The part of the application that allows for connection to the server

Usage¶

class mmmoc.MMOC()¶

The MMOC object is the basis for connecting to the server. It acts as the centerpoint for all multiplayer actions, such as sending data and getting data.

mmmoc.MMOC.init(id_len=8, wsurl="//" + document.domain + ":" + location.port)¶

Initialize the player’s ID and connection to the main server

Arguments:
  • id_len (number) – Length of the id to be generated
  • wsurl (string) – WebSockets URL to connect to
mmmoc.MMOC.sendData()¶

Send the player’s coordinates, ID and other data to the server

mmmoc.MMOC.getData()¶

Get every players’ as an object

Returns:each player’s coordinates, id and other data
Return type:object
mmmoc.MMOC.changeX(by)¶

Change the player’s X value by a specified value

Arguments:
  • by (number) – value to change X by
mmmoc.MMOC.changeY(by)¶

Change the player’s Y value by a specified value

Arguments:
  • by (number) – value to change Y by
mmmoc.MMOC.setOther(key, value)¶

Set a piece of data to be broadcasted to every other player. The data will be located at the key key

Arguments:
  • key (string) – key to store the data at
  • value (any) – value to be stored
mmmoc.MMOC.isconnected()¶

Check if the client has connected to the server

Returns:promise that will resolve to a boolean value
Return type:Promise

Example¶

let mp = new MMOC();
mp.init();

while (true):
    if(...) {
        mp.changeX(1);
    }
    if(...) {
        mp.changeX(-1);
    }
    if(...) {
        mp.changeY(1);
    }
    if(...) {
        mp.changeY(-1);
    }

    let d = mp.getData();
    drawToScreenFunction(d);

To view a more flushed out example see main.js in the example.
