Client¶

The part of the application that allows for connection to the server

Usage¶

class mmoc.MMOC()¶

The MMOC object is the basis for connecting to the server. It acts as the centerpoint for all multiplayer actions, such as sending data and getting data.

mmoc.MMOC.init(id_len=8, wsurl="//" + document.domain + ":" + location.port)¶

Initialize the player’s ID and connection to the main server

Arguments:
  • id_len (number) – Length of the id to be generated
  • wsurl (string) – WebSockets URL to connect to
mmoc.MMOC.sendPlayerData()¶

Send the player’s coordinates, ID and other data to the server

mmoc.MMOC.sendObjectData(object)¶

Send a projectile’s or any moving object’s data that is not a player

Arguments:
  • object (MovingObject) – The object that is having its data being sent
mmoc.MMOC.removeObject(object)¶

Remove an object from the server

Arguments:
  • object (MovingObject) – The object to be removed
mmoc.MMOC.getPlayers()¶

Get every player as an object

Returns:each player’s coordinates, id and other data
Return type:object
mmoc.MMOC.getGlobals()¶

Get all of the server globals

Returns:all of the server’s global variables set in config.yml
Return type:object
mmoc.MMOC.getObjects()¶

Get all of the moving objects

Returns:all of the moving objects
Return type:object
mmoc.MMOC.changeX(by)¶

Change the player’s X value by a specified value

Arguments:
  • by (number) – value to change X by
mmoc.MMOC.changeY(by)¶

Change the player’s Y value by a specified value

Arguments:
  • by (number) – value to change Y by
mmoc.MMOC.changeZ(by)¶

Change the player’s Z value by a specified value

Arguments:
  • by (number) – value to change Z by
mmoc.MMOC.changeOrientation(by)¶

Change the player’s orientation by a specified value

Arguments:
  • by (nubmer) – value to change the orientation by
mmoc.MMOC.setOther(key, value)¶

Set a piece of data to be broadcasted to every other player. The data will be located at the key key

Arguments:
  • key (string) – key to store the data at
  • value (any) – value to be stored
mmoc.MMOC.isconnected()¶

Check if the client has connected to the server

Returns:promise that will resolve to a boolean value
Return type:Promise


class mmoc.MovingObject(mesh, p, r)¶

The MovingObject class is for creating new objects that need to be moved and synced between clients

Arguments:
  • object (mesh) – the mesh/data for the object
  • function (r) – a function for getting the position of the mesh. Must modify this.x, this.y, and this.z
  • function – a function for rendering the mesh
mmoc.MovingObject.render()¶

Render the mesh

mmoc.MovingObject.setOther(key, value)¶

Set a property for the object

Arguments:
  • string (key) – the key for accessing the object
  • object (value) – a json serializable object
mmoc.MovingObject.getOther(key)¶

Get a property at the key ‘key’

Arguments:
  • string (key) – the key for accessing the value
Returns:

the value at the specified key
Return type:

object

Example¶

let mp = new MMOC();
mp.init();

while (true):
    if(...) {
        mp.changeX(1);
    }
    if(...) {
        mp.changeX(-1);
    }
    if(...) {
        mp.changeY(1);
    }
    if(...) {
        mp.changeY(-1);
    }

    let d = mp.getData();
    drawToScreenFunction(d);

To view a more flushed out example see main.js in the example.
