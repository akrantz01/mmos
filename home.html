Welcome to MMOS¶

MMOS is the simple solution for creating a MMO game in JavaScript. Click here to download the most recent version.

Parts¶

  • Server - handle connected clients
    • the life blood of the system
    • handles communications between clients
    • stores all the data
  • Client - connect to the server
    • communicates with the server
    • retrieves all the data
  • Configuation - configure the server
    • set upstreams
    • set debug credentials
    • set global variables

Basic Setup¶

  1. Download the server
  2. Run the server with ./server [configfile.yml]
  3. Download the client
  4. Implement into your game
  5. Try it out
