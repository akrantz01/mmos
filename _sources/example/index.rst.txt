Simple Example
==============
Demonstrates the basic usage with colored balls. `Click here <https://github.com/akrantz01/mmos/tree/master/example>`_ to download the example.

Directory Structure
-------------------
::

    public/
        main.css
        main.js
        mmoc.js
        index.html
    config.yml
    server

View the Code
-------------
* :doc:`js`
* :doc:`html`
* :doc:`css`
* :doc:`yml`

Running the Server
------------------
#. Make sure you have the ``server`` downloaded from `here <https://github.com/akrantz01/mmos/releases/latest>`_
#. Run the server with ``./server config.yml``
#. Navigate to `localhost:5000 <http://localhost:5000>`_
